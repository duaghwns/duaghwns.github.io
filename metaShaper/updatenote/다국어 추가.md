# 다국어 지원 버전 (Multilingual Version)

MetaShaper에 4개 언어 지원을 추가한 버전입니다.

## 지원 언어

- 🇰🇷 한국어 (Korean) - KR
- 🇺🇸 English - US
- 🇨🇳 中文 (Chinese) - CN
- 🇯🇵 日本語 (Japanese) - JP

## 파일 구조

```
multilang-version/
├── index.html       # 메인 HTML (i18n.js 먼저 로드)
├── styles.css       # 스타일시트
├── i18n.js          # 다국어 번역 데이터만 포함 (NEW!)
├── raw.js           # 메인 JavaScript (번역 데이터 제외)
└── README.md        # 문서
```

## 주요 변경사항

### ✨ 번역 데이터 분리
- **i18n.js**: 모든 언어의 번역 데이터만 포함
- **raw.js**: 메인 로직만 포함 (170줄 이상 감소!)
- 파일 관리가 쉬워지고 번역 추가/수정이 간편함

### 🎨 UI 개선
- 언어 선택: 셀렉트 박스 (KR/US/CN/JP)
- 화살표 아이콘 제거로 깔끔한 디자인
- font-weight: 600, letter-spacing으로 가독성 향상

## 주요 기능

### 1. UI 다국어 지원
- 헤더 우측에 언어 선택 셀렉트 박스
- 버튼, 라벨, 메시지 등 모든 UI 텍스트 자동 변경
- 선택한 언어는 localStorage에 저장

### 2. GPS 위치 정보 다국어
- OpenStreetMap Nominatim API 활용
- 선택한 언어에 따라 국가명/도시명 표시
- 예시:
  - KR: "대한민국 서울특별시 강남구"
  - US: "South Korea Seoul Gangnam-gu"
  - CN: "韩国 首尔特别市 江南区"
  - JP: "韓国 ソウル特別市 江南区"

### 3. 언어 전환 시 자동 업데이트
- UI 텍스트 즉시 변경
- GPS 좌표가 있으면 위치 정보도 자동 갱신

## 구현 세부사항

### i18n.js (NEW!)
```javascript
const translations = {
    ko: { appTitle: 'MetaShaper', upload: '이미지 업로드', ... },
    en: { appTitle: 'MetaShaper', upload: 'Upload Image', ... },
    zh: { appTitle: 'MetaShaper', upload: '上传图片', ... },
    ja: { appTitle: 'MetaShaper', upload: '画像アップロード', ... }
};
```

### raw.js
```javascript
// 번역 함수 (translations는 i18n.js에서 불러옴)
function t(key) {
    return translations[currentLang][key] || key;
}
```

### index.html
```html
<!-- i18n.js를 먼저 로드해야 함 -->
<script src="i18n.js"></script>
<script src="raw.js"></script>
```

### CSS - 언어 선택 셀렉트 박스
```css
.lang-select {
    appearance: none;  /* 기본 화살표 제거 */
    background-image: none;
    font-weight: 600;
    letter-spacing: 0.5px;
    min-width: 50px;
}
```

## 번역 추가 방법

새로운 언어를 추가하려면 `i18n.js` 파일만 수정하면 됩니다:

```javascript
const translations = {
    ko: { ... },
    en: { ... },
    zh: { ... },
    ja: { ... },
    // 새 언어 추가
    fr: {
        appTitle: 'MetaShaper',
        upload: 'Télécharger une image',
        // ... 나머지 번역
    }
};
```

그리고 HTML의 셀렉트 박스에 옵션 추가:
```html
<select id="langSelect" class="lang-select" onchange="changeLanguage(this.value)">
    <option value="ko">KR</option>
    <option value="en">US</option>
    <option value="zh">CN</option>
    <option value="ja">JP</option>
    <option value="fr">FR</option> <!-- 새 옵션 -->
</select>
```

## 장점

### 📦 파일 크기 최적화
- raw.js에서 170줄 이상 제거
- 번역 데이터를 별도 파일로 분리
- 필요시 i18n.js만 수정/교체 가능

### 🔧 유지보수성
- 번역 추가/수정이 쉬움
- 로직과 데이터의 분리
- 여러 개발자가 동시 작업 가능

### 🚀 확장성
- 새로운 언어 추가 간편
- 번역 데이터를 서버에서 동적으로 로드 가능
- 다양한 번역 전략 적용 가능

## 테스트 방법

1. `multilang-version/index.html` 열기
2. 헤더 우측의 언어 선택 박스 클릭
3. KR/US/CN/JP 선택하며 UI 변화 확인
4. GPS 정보 포함 이미지 업로드
5. 언어 변경 시 위치 정보 자동 갱신 확인

## 적용 방법

테스트 후 문제가 없다면:
1. `i18n.js` (새 파일!) 추가
2. `raw.js` 변경사항 적용
3. `styles.css` 변경사항 적용
4. `index.html` 변경사항 적용 (i18n.js 로드 추가)
5. 빌드 프로세스 조정 (i18n.js 포함)
