# 모바일 재정렬 버튼 테스트 버전

모바일 화면에서 메타데이터 순서를 변경할 수 있도록 화살표 버튼을 추가한 테스트 버전입니다.

## 변경사항

### 1. JavaScript (raw.js)
- `moveItemUp(index)`: 항목을 위로 이동하는 함수 추가
- `moveItemDown(index)`: 항목을 아래로 이동하는 함수 추가
- `renderMetadataList()`: 각 항목에 위/아래 화살표 버튼 추가

### 2. CSS (styles.css)
- `.mobile-reorder-btns`: 화살표 버튼 컨테이너 스타일
- `.mobile-arrow-btn`: 화살표 버튼 스타일
- 모바일 화면(768px 이하)에서:
  - 화살표 버튼 표시
  - 드래그 핸들 숨김

### 3. HTML (index.html)
- `app.js` → `raw.js`로 변경

## 기능

### 데스크톱
- 기존과 동일하게 드래그 앤 드롭으로 순서 변경

### 모바일
- ⬆️ 위로 버튼: 항목을 위로 이동
- ⬇️ 아래로 버튼: 항목을 아래로 이동
- 첫 번째 항목의 위로 버튼, 마지막 항목의 아래로 버튼은 비활성화

## 테스트 방법

1. `mobile-reorder-test/index.html` 파일을 브라우저로 열기
2. 개발자 도구에서 모바일 화면으로 전환 (768px 이하)
3. 설정 탭에서 화살표 버튼으로 항목 순서 변경 테스트

## 적용 방법

테스트 후 문제가 없다면:
1. `raw.js`의 변경사항을 원본 `raw.js`에 적용
2. `styles.css`의 변경사항을 원본 `styles.css`에 적용
3. 빌드 프로세스를 통해 `app.js` 재생성
